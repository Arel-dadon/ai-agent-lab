# Testing Structure

This directory contains the test suites for the ai-agent-lab project.

## Structure

- `tests/`: Root directory for all tests.
- `tests/test_smoke.py`: Basic smoke tests to verify the testing environment.

## Running Tests

To run all tests, use the following command from the root of the repository:

```bash
pytest
```

To run a specific test file:

```bash
pytest tests/test_smoke.py
```
