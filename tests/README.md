# Tests

This directory contains the test suite for the ai-agent-lab project.

## Structure

- **Unit Tests**: Test individual components or functions in isolation.
- **Integration Tests**: Test the interaction between different components or services.
- **End-to-End (E2E) Tests**: Test the entire application flow from the user's perspective.

## Running Tests

To run the tests, use the following command:

```bash
pytest
```

Or using the standard unittest module:

```bash
python -m unittest discover tests
```
